Instructions

Select pupper from the menu.

Activate the controller. This is like pairing a joystick.

Activate minipupper. This is like pressing L1 on a joystick.

Select walk from the menu.

The start and stop buttons toggle the run mode, like pressing R1 on a joystick.

When the robot is not running the sliders will set the values that are used by the robot when the start button is pressed.

When the robot is running the values of a slider will be sent to the robot.

When the stop button is pressed the robot will be set to the neutral position.
